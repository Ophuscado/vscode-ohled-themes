# OhLED Themes (142-in-1) for VSCode

## About

The OhLED Themes (142-in-1) for VSCode are inspired by the classic CRT monitors of yesteryear, specially crafted for the modern OLED displays of today. It includes an optional timer to enable an automatic colour cycle in order to further avoid screen burning.
